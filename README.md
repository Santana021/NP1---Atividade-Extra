Projeto Desenvolvimento Web - NP1

Este repositório contém um site institucional desenvolvido para a disciplina de Desenvolvimento Web, com foco em semântica HTML5, CSS3 e responsividade. O projeto foi finalizado para a entrega da Atividade Extra no dia 30/03/2026.

## Tecnologias e Técnicas Aplicadas

O desenvolvimento seguiu os requisitos obrigatórios da avaliação:
* **Elementos HTML5:** Uso de tags semânticas como header, main, footer, tabelas, listas e links internos/externos.
* **Mobile First:** Estilização iniciada para dispositivos móveis com evolução para telas maiores via Media Queries.
* **Sistemas de Layout:** Implementação de Flexbox para alinhamentos e barras de navegação, e CSS Grid para a estrutura principal.
* **CSS Custom Properties:** Definição de paleta de cores através de variáveis no :root.
* **Unidades Relativas:** Utilização de rem, em e porcentagem para tipografia e blocos responsivos.
* **Breakpoints:** Implementação de 4 pontos de interrupção: 480px, 768px, 1024px e 1200px.

## Estrutura de Arquivos

* **index.html:** Página principal com informações do trabalho.
* **produtos.html:** Catálogo contendo a tabela estruturada de registros.
* **style.css:** Folha de estilo externa contendo a lógica visual e responsiva.
* **logo.png:** Identidade visual utilizada no projeto.

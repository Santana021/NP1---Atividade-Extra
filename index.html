Sobre o trabalho

Esse site foi todo desenvolvido aplicando os conceitos de marcação semântica e estilização (CSS3) com conceito Mobile First pedido em sala.

Principais aprendizados:

  • Uso correto de tags semânticas (header, main, footer, section)
  • Estilização usando variáveis no :root no CSS
  • Responsividade com media queries
